# Add me on SkyCards
Add a button to your project for users to add you on skycards.

# Usage
simply insert this into your project:
<script 
  src="https://letsplaysimi.github.io/add-me-on-skycards/widget.js" 
  data-name="YourName" 
  data-code="1234 5678 9123">
</script>
