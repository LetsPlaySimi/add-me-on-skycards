# Add me on SkyCards
Add a button to your project for users to add you on skycards.
![animation](https://github.com/LetsPlaySimi/add-me-on-skycards/blob/main/animation.gif?raw=true)

# Usage
Simply import this script:
```
<script src="https://letsplaysimi.github.io/add-me-on-skycards/widget.js"></script>
```
Then you can place the button wherever you want:
```
<sky-cards name="YourName" code="1234 5678 9123"></sky-cards>
```
![sky-cards](https://github.com/LetsPlaySimi/add-me-on-skycards/blob/main/sky-cards.png?raw=true)<br>
or
```
<sky-cards-compact name="YourName" code="1234 5678 9123"></sky-cards-compact>
```
![sky-cards-compact](https://github.com/LetsPlaySimi/add-me-on-skycards/blob/main/sky-cards-compact.png?raw=true)
